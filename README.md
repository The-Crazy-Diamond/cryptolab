# Cryptolab

Cryptolab is a Python/sagemath CLI tool for experimenting with classical cryptography:
- Ciphering / deciphering texts
- Cryptanalysis tools (frequency analysis, Kasiski, etc.)

This project is a way for me to develop programming skills while having good time on cryptology which is of great interest to me.

## Features

- Caesar cipher (planned)
- Monoalphabetic substitution using frequency analysis (planned)
- Vigenère cipher (planned)
- Other polyalphabetic substitution methods (planned)
- Enigma (planned)
- language recognition/automatic (planned)
